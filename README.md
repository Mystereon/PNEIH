# PNEIH
Plasma Neuron Entity in HTML
An attempt to create a mind with a tone responsive 30 x 30 analogue neural grid 
that behaves as a parrot would 
https://plasmaent-jcd7ga28.manus.space/plasma.html

900 analogue-style neurons
sound + motion perception
floating excitation states
colour mood behaviour
chirp synthesis
reasoning feedback panel
responsive mobile/desktop layout
transparent internal state logging

It does not behave like a human.
It behaves like a tiny perceptual field trying to understand its environment.

This is the beginning of the engine.
